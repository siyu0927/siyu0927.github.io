<h1>打地鼠</h1>
<p>這篇文章會介紹，如何在 Scratch 裡使用重複無限次、分身、更換造型、偵測滑鼠按下、隨機數...等積木，實作打地鼠的小遊戲。</p>
<blockquote>
<p>相關文章參考：<a href="/category/scratch/code/control.html#a2" title="重複" target="_blank">重複</a>、<a href="/category/scratch/code/sensing.html#a3" title="偵測滑鼠按下" target="_blank">偵測滑鼠按下</a>、<a href="/category/scratch/code/looks.html#a2" title="更換造型" target="_blank">更換造型</a>、<a href="/category/scratch/code/control.html#a5" title="建立分身" target="_blank">建立分身</a>。</p>
</blockquote>
<h2 id="a1">角色與背景設定 <a href="#a1" title="角色與背景設定 "></a></h2>
<p>繪製角色前，先更換舞台的背景為星空，並在背景裡，用「圓形」繪製六個空心橢圓形，上方輸入「打地鼠小遊戲」的文字。</p>
<blockquote>
<p>新增圖片背景會是「點陣圖」，轉換成「向量圖」之後就能使用向量的繪圖功能 ( 參考：<a href="/category/scratch/sprite/costumes.html" title="繪製造型＆背景" target="_blank">繪製造型＆背景</a>  )。</p>
</blockquote>
<p><img attr-src="/webp/scratch/example/hamster-hit-01.webp" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" class="b1" style="width:622px; height:411px;" alt="Scratch 教學 - 打地鼠"></p>
<p>舞台背景完成後，回到角色編輯區，刪除貓咪角色，加入「鬼魂 ghost」的角色 ( 參考：<a href="/category/scratch/info/interface.html#a9" title="Scratch 角色設定" target="_blank">Scratch 角色設定</a> )。</p>
<p><img attr-src="/webp/scratch/example/hamster-hit-02.webp" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" class="b1" style="width:397px; height:234px;" alt="Scratch 教學 - 打地鼠"></p>
<p>鬼魂角色總共有四種造型，待會會使用 ghost-a ( 正常表情 ) 和 ghost-c ( 驚訝表情 ) 兩種。</p>
<p><img attr-src="/webp/scratch/example/hamster-hit-03.webp" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" class="b1" style="width:340px; height:442px;" alt="Scratch 教學 - 打地鼠"></p>
<h2 id="a2">積木程式原理 <a href="#a2" title="積木程式原理 "></a></h2>
<p>透過「定位」和「建立分身」的積木，建立六個鬼魂，分別放在背景對應的橢圓圈內。</p>
<p><img attr-src="/webp/scratch/example/hamster-hit-04.webp" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" style="width:626px; height:527px;" alt="Scratch 教學 - 打地鼠"></p>
<p>建立分身後，使用「隱藏」將角色本身隱藏，接著「顯示」每個建立的分身，並透過「<em>圖像效果</em>」隨機改變分身的顏色，完成後點擊綠旗，應該就能看到六個不同顏色的鬼魂出現在舞台中。( 參考：<a href="/category/scratch/code/looks.html#a5" title="圖像效果" target="_blank">圖像效果</a> )</p>
<p><img attr-src="/webp/scratch/example/hamster-hit-05.webp" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" style="width:546px; height:567px;" alt="Scratch 教學 - 打地鼠"></p>
<p>將分身的「隱藏」，換成「重複無限次」、「顯示」、「等待」和「隱藏」的組合，就能<em>讓鬼魂角色在不同時間出現，出現兩秒後會消失</em>。( 因為打地鼠的老鼠，也是會在探頭出現之後，過了幾秒就縮頭消失 )</p>
<p><img attr-src="/webp/scratch/example/hamster-hit-06.webp" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" style="width:531px; height:567px;" alt="Scratch 教學 - 打地鼠"></p>
<p>完成後點擊綠旗，就能看到六個不同顏色的鬼魂在不同時間出現，過兩秒後又消失，在舞台中不斷重複這個過程。</p>
<p><img attr-src="/webp/scratch/example/hamster-hit-07.gif" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" style="width:434px; height:332px;" alt="Scratch 教學 - 打地鼠"></p>
<p><em>再度建立</em>一個「當分身產生」的積木，使用重複無限次積木，判斷滑鼠點擊 ( <em>分身同時要碰到鼠標以及滑鼠鍵被按下</em> )，就更換為驚訝的造型，等待 0.3 秒之後隱藏分身，並將分身恢復原本正常造型。</p>
